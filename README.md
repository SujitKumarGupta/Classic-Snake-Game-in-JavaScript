# Classic Snake Game in JavaScript
This is a browser-based version of the classic Snake Game, developed using HTML, CSS, and JavaScript. The game is rendered on a grid system, and the snake is controlled using keyboard arrow keys. It features real-time animations, collision detection, score tracking, and local storage for high score persistence.

Key Features:

🎮 Interactive Gameplay: Snake moves in response to arrow key inputs with smooth animation.

🍎 Food Mechanics: Snake grows longer each time it eats randomly placed food items.

💥 Collision Detection: Game ends when the snake collides with walls or itself.

🎵 Sound Effects: Audio feedback for movement, eating food, and game over events.

🏆 High Score System: Uses localStorage to retain high scores across sessions.

🧠 Efficient Logic: Game loop managed using requestAnimationFrame() for optimized performance.
